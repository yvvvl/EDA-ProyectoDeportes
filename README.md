# EDA Proyecto Deporte

## Descripción del Proyecto
Este proyecto se centra en el análisis de datos deportivos utilizando Jupyter Notebooks y Python. El objetivo es proporcionar insigths a partir de datos de rendimiento deportivo, que pueden ser utilizados por entrenadores y analistas.

## Análisis de Datos
Se utilizarán Jupyter Notebooks para llevar a cabo el análisis de datos. Los scripts en Python realizarán diversas operaciones sobre los datos, permitiendo la visualización y comprensión de patrones.

## Instrucciones de Instalación
1. Clona el repositorio:
   ```bash
   git clone https://github.com/yvvvl/EDA-ProyectoDeportes.git
   ```
2. Navega a la carpeta del proyecto:
   ```bash
   cd EDA-ProyectoDeportes
   ```
3. Instala las dependencias:
   ```bash
   pip install -r requirements.txt
   ```

## Estructura del Proyecto
- `notebooks/` - Notebooks de Jupyter con análisis en profundidad.
- `data/` - Conjunto de datos utilizados para el análisis.
- `src/` - Scripts en Python para el procesamiento y análisis de datos.

Esperamos que encuentres útil este proyecto y que te ayude a profundizar en el análisis de datos deportivos.
